Project 2: Theater Inventory System (Hash Tables)

← Back to Resume

Description

This project implemented a theater inventory system in the C programming language. The system uses a hash table with separate chaining to store items like popcorn and soda. It allows buying supplies, selling items, updating prices, and tracking total cash and operation complexity. The project simulated how theaters manage inventory and transactions.
